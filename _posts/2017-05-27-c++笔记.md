#### static_cast

>用法：static_cast < type-id > ( expression )
<br>
>C++中的static_cast执行非多态的转换，用于代替C中通常的转换操作
<br>

```c++
double da = 1.1;
void *pa = &da;
double *dp = static_cast<double*>(pa);
int ia = static_cast<int>(da);
cout << *dp << endl;
cout << ia << endl;
```
